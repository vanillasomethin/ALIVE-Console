# Contributing

Thanks for helping improve the ALIVE Signage Console! This repository uses a standard Next.js workflow.

## Prerequisites
- Node.js 18+
- npm (bundled with Node)

## Install
```bash
npm install
```

## Run locally
```bash
npm run dev
```
Then open http://localhost:3000.

## Build
```bash
npm run build
```

## Start (production)
```bash
npm run start
```

## Lint
```bash
npm run lint
```

## Testing
No automated tests are configured yet. If you add tests, please document how to run them here.
